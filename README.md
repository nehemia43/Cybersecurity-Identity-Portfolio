## 🗂️ Portfolio Architecture

### 📁 [06-Lifecycle-Governance-Automation (AI-Enhanced)](./06-Lifecycle-Governance-Automation)
**Focus:** Healthcare JML (Joiner/Mover/Leaver), **AI Risk Scoring**, Automated Offboarding.
* **Business Value:** Architected a **"Zero-Touch" Deprovisioning Framework** to address HIPAA/NIST compliance for immediate access revocation.
* **Technical Execution:** Developed a Python-driven lifecycle engine that synchronizes HR authoritative sources with **Microsoft Entra ID**.
* **Innovation:** Integrated an **AI-driven Governance Layer** to identify "Toxic Access Combinations" and Separation of Duty (SoD) violations during termination events.

### 📁 [05-Identity-System-Administration](./05-Identity-System-Administration)
**Focus:** Domain Migrations, Microsoft Graph API, Identity Mapping.
* **Business Value:** Engineered a **Domain Migration Impact Analyzer** to facilitate zero-downtime enterprise transitions.
* **Technical Execution:** Automated the discovery of high-risk SAML application dependencies and mapped mutable UPNs to immutable Object IDs for 10,000+ identities.

### 📁 [03-PAM-Privileged-Access](./03-PAM-Privileged-Access)
**Focus:** CyberArk (NuSil), Privileged Session Manager (PSM), Credential Hygiene.
* **Business Value:** Secured privileged "Administrative Access" through automated vaulting and isolated session proxying.
* **Technical Execution:** Automated 45-day credential rotation and implemented Auto-Discovery via LDAP to secure "Shadow Admin" accounts in ITAR-regulated environments.

### 📁 [01-IGA-Identity-Governance](./01-IGA-Identity-Governance)
**Focus:** SailPoint, Role Mining, Birthright Provisioning.
* **Technical Execution:** Engineered automated Joiner/Mover/Leaver (JML) workflows and ABAC policies to ensure **Least Privilege** from Day 1.
