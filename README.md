# 🛡️ Enterprise Identity & Access Management (IAM) Architecture Portfolio

Welcome to my master portfolio. This repository contains **42 hands-on technical labs** and **10 GRC projects** showcasing enterprise-grade architecture, deployment, and automation across the three core pillars of modern Identity Security: **IGA, AM, and PAM**.

## 🎯 Objective
To demonstrate advanced technical execution and compliance governance through the implementation of zero-trust architectures, automated lifecycle management, and privileged session isolation.

## 🗂️ Repository Structure

### 📁 [01-IGA-Identity-Governance](./01-IGA-Identity-Governance)
**Focus:** SailPoint IdentityIQ, Lifecycle Management (JML), Role Mining, Birthright Provisioning.
* **Key Highlights:** * Engineered automated Joiner/Mover/Leaver (JML) workflows utilizing HR authoritative sources.
  * Executed intelligent role mining campaigns to consolidate granular entitlements into IT Access Profiles.
  * Implemented Attribute-Based Access Control (ABAC) for automated birthright provisioning.

### 📁 [02-AM-Access-Management-SSO](./02-AM-Access-Management-SSO)
**Focus:** Okta, Microsoft Entra ID, SAML 2.0, OpenID Connect (OIDC), MFA.
* **Key Highlights:**
  * Architected cross-platform federation and Single Sign-On (SSO) deployments.
  * Configured SCIM provisioning and adaptive multi-factor authentication policies.
  * Integrated custom enterprise applications using standardized OAuth 2.0/OIDC flows.

### 📁 [03-PAM-Privileged-Access](./03-PAM-Privileged-Access)
**Focus:** CyberArk, Privileged Session Manager (PSM), Central Policy Manager (CPM).
* **Key Highlights:**
  * Automated credential hygiene including 45-day rotation and domain-level reconciliation.
  * Engineered PSM proxies for isolated RDP/SSH sessions, enforcing Dual-Control authorization.
  * Implemented Auto-Discovery via LDAP to detect, vault, and secure "Shadow Admin" accounts.

### 📁 [04-Cloud-Identity-IAM](./04-Cloud-Identity-IAM)
**Focus:** Non-Human Identity (NHI) Governance, Python/PowerShell Automation, API Integration.
* **Key Highlights:**
  * Developed custom Python auditors to discover and remediate orphaned service accounts.
  * Engineered "Zero-Touch" identity lifecycle pipelines using JavaScript and Okta APIs.
  * Automated "Audit Readiness" reporting aligned with NIST CSF and ISO 27001 standards.
